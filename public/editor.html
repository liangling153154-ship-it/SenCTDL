⚠️ Editor đang chạy bằng file:// — iframe sẽ bị chặn. Mở qua Live Server (chuột phải file → "Open with Live Server").
Bật Command Mode ở topbar, rồi click 1 element trên trang để chọn.
Edit Mode: click text → sửa tại chỗ.
Command Mode: click element → viết prompt mô tả ý định (rewrite / translate / add new / remove…). Claude sẽ thực hiện.
Breadcrumb: click ancestor để mở rộng vùng chọn (vd: chọn cả card thay vì chỉ heading).
Esc bỏ chọn · Enter lưu text edit.
