# Sessions Log — Điều Hành Du Lịch

Thư mục này lưu lại lịch sử làm việc theo từng phiên.

Mục đích: để **Antigravity** (IDE) và **Claude Code** (VSCode) có thể co-op — đọc context của nhau, tránh làm trùng, và biết tiếp theo cần làm gì.

---

## Quy ước đặt tên file

```
YYYY-MM-DD_N_agent.md
```

Ví dụ:
- `2026-06-28_1_claude-code.md` — phiên đầu tiên ngày 28/06, do Claude Code thực hiện
- `2026-06-29_1_antigravity.md` — phiên đầu tiên ngày 29/06, do Antigravity thực hiện

**Agent viết tắt:**
- `claude-code` — Claude Code trong VSCode
- `antigravity` — Antigravity trong IDE

---

## Template cho mỗi phiên

Xem file `_TEMPLATE.md` trong thư mục này.
